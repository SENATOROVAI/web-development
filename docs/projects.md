# Проекты

## Проект 1: Study Planner

> Веб-приложение для планирования учебных задач и дедлайнов.

- **Описание:** помогает структурировать учебный график и следить за прогрессом.
- **Технологии:** HTML, CSS, JavaScript, LocalStorage.
- **Репозиторий:** [GitHub](https://github.com/)

---

## Проект 2: Notes API

> Учебный backend-сервис для управления заметками.

- **Описание:** REST API для создания, редактирования и удаления заметок.
- **Технологии:** Python, FastAPI, SQLite.
- **Репозиторий:** [GitHub](https://github.com/)
